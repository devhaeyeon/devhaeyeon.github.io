개발 문서에서 'behind the scene'의 뜻

서론

도커 공식 홈페이지에서 샘플 튜토리얼을 번역하다가 난관에 부딪혔다.
‘뒤에서 장면이?’
‘장면이 뒤에서?’
‘뒤에서 실행이 되고 있다?’
‘음 ? 무엇을’
…
수 많은 해석들을 했지만, 번역을 할 수 없었다.
…

의미

사전적인 의미 : ‘무대 뒤에서 / 이면에서’
개발 문서에서 번역할 때 대략적인 의미로는 ‘내부적으로 어떤 어떤 일이 일어난다.’
약간 ‘내부적인’ 것이라고 보면 될 것 같다.
